<h1 align="center">Jhon Paul Baonil's Personal Portfolio</h1>

<h3 align="center">A cyber-themed, modern, and futuristic personal portfolio website for Jhon Paul Baonil.</h3>

---

## Features
- **Cyber / Professional Theme**: Dark UI with neon cyan and purple accents.
- **Animations**: Custom `h1` text typing animations, scroll-trigger reveal effects, and looping rainbow borders.
- **Glassmorphism**: Premium "watered glass" frosted aesthetics.
- **Multi-page Structure**: Separated HTML files (Home, About Me, Projects, Contact) in the `html` folder.
- **Responsive & Accessible**: Fully responsive layouts optimized for mobile and desktop, featuring high-contrast readability.
- **Project Showcase**: Pre-configured placeholder interfaces for hackathon events and portfolio entries.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- Vanilla JavaScript

## Setup & Running Locally
1. Clone or download the repository.
2. Open `index.html` in your web browser (it will redirect you to `html/home.html`), or navigate directly to `html/home.html`.
3. The site does not require a local backend server to view, although running it via a local server like `Live Server` is recommended to prevent relative path issues in some browsers.
